# Mandatory Handoff Contract

Use this payload for every cross-agent delegation.

## Required Fields
1. **Objective**
2. **Scope** / **Out-of-scope**
3. **Acceptance Criteria** (atomic and testable)
4. **Dependencies**
5. **Risks / assumptions**
6. **Required artifacts** (docs, code, tests, evidence)
7. **Deadline / SLA**
8. **Reviewer(s) / approver(s)**
9. **Traceability links** (task, PR, ADR, test report)

## Delegation Template
```md
### Handoff

- Scope:
```

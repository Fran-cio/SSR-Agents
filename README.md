# SSR-Agents

Snapshot repository for current ClawCode/OpenClaw configuration.

## Included
- Workspace root markdown/config files (`clawcode/workspace-root/`)
- Full multi-agent definitions (`clawcode/agents/<role>/`)
- Agent relationship map (`clawcode/AGENT_RELATIONSHIPS.md`)
- Agent RACI matrix (`clawcode/AGENT_RACI_MATRIX.md`)
- Operating model and gates (`clawcode/OPERATING_SYSTEM.md`, `clawcode/GATE_POLICY.md`)
- Mandatory handoff contract (`clawcode/HANDOFF_CONTRACT.md`)
- Team launch plan (`clawcode/TEAM_LAUNCH_PLAN.md`)
- Installed and curated external skills (`clawcode/skills/`)
- skills.sh curation notes (`clawcode/SKILLS_CURATION.md`)
- File inventory (`INVENTORY.md`)

## Source
Exported from active orchestrator workspace and installed OpenClaw skills.
